Portafolio

    Drop Aid 2022

    • Trabajo para el ramo Taller de Interacción III


    Profesores: Germán Espinoza y Mauricio Reyes

    Equipo: Mariajosé Mendoza y Constanza González

    Descripción del encargo: Detectar un problema oportunidad, creando una propuesta de valor utilizando design thinking, indicando una hipótesis tecnológica y de mercado (usuario). Definiendo detalles tecnológicos, con un desarrollo de prototipo funcional de calidad y pertinencia, apoyado de una memoria visual descriptiva de cómo fue el proceso de diseño 3D, 2D y códigos empleados.

    Reconocimiento


    Con este proyecto participamos en el Concurso Impacto Emprendedor 2022 quedando seleccionado entre los 25 mejores de un total de 500 participantes.

    Proceso de Diseño


    El primer paso fue detectar una oportunidad o necesidad no resuelta dentro del mercado, utilizando la metodología Design Thinking. Este enfoque nos permitió profundizar en la experiencia del usuario y generar una propuesta de valor única, orientada a solucionar un problema real.

    A partir de este análisis, formulamos una hipótesis tecnológica y de mercado basada en el comportamiento y necesidades del usuario. Nos preguntamos: ¿Cómo podríamos usar la tecnología para mejorar esta experiencia y satisfacer estas demandas de manera eficiente?

    Problema detectado


    El transporte público en Santiago presenta una creciente preocupación para jóvenes universitarias que enfrentan situaciones de acoso.

    Estas situaciones generan ansiedad y vulnerabilidad, lo que lleva a la búsqueda de soluciones que mejoren la seguridad y la respuesta rápida ante una situación de peligro.

    Propuesta


    Drop Aid es un sistema de seguridad colaborativa diseñado para mujeres que viajan solas. El proyecto ofrece una pulsera y un anillo inteligentes que, al ser activados, emiten una vibración y envían una señal a una aplicación móvil. La app notifica a otras usuarias cercanas y a los contactos de emergencia predefinidos.

    Con este sistema, las jóvenes pueden alertar y ser asistidas rápidamente, creando una red de apoyo.

    Prototipado en 3D y Diseño


    Utilizamos Fusion 360 para diseñar el render de la pulsera, que posteriormente fue impresa en 3D para lograr un prototipo funcional. Este proceso permitió visualizar cómo sería el producto final y hacer ajustes de ergonomía y estilo.

    Desarrollo Tecnológico


    Implementamos un mini motor de vibración y un módulo ESP8266, que fueron integrados con Arduino para crear la funcionalidad principal del dispositivo. La pulsera vibra al ser activada y envía la señal de alerta mediante el sistema conectado.

    Conectividad


    En este proyecto, Blynk y IFTTT fueron esenciales para crear el sistema de notificaciones. Blynk permitió conectar el módulo ESP8266 con el botón de la pulsera, enviando una señal cuando se activaba. IFTTT automatizó las respuestas, como enviar alertas a los contactos de emergencia, facilitando una solución rápida y funcional para la notificación en situaciones de acoso.
